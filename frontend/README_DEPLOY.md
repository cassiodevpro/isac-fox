# Deploy no Vercel

1. Faça login em https://vercel.com/
2. Clique em "New Project" e conecte o repositório deste projeto.
3. Confirme que o diretório do projeto é `frontend`.
4. O Vercel detecta automaticamente projetos Next.js.
5. Se necessário, configure variáveis de ambiente usando o arquivo `.env.example` como referência.
6. Clique em "Deploy" e aguarde a publicação.

Pronto! Seu site estará disponível em uma URL do Vercel.
